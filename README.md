## 😎 개요

- 사용자의 선택에 따라 어울리는 음악 장르를 추천해줍니다.
- 선택된 장르에 해당하는 음악을 들려줍니다.
- MBTI 성격유형에 기반하여 사용자의 성격을 분석해줍니다.

## 😊 기술스택

- HTML/CSS
- JavaScript
- jQuery
- media query
